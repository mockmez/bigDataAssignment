# bigDataAssignment
Big Data Assignment

Group project for COMP30770
Programming for Big Data


Data sources are

| File Name                  | Description                          | Link                                                                                      |
|----------------------------|--------------------------------------|-------------------------------------------------------------------------------------------|
| Books.csv                  | Contains book information (title, author, etc.) | [Books.csv](https://www.kaggle.com/datasets/saurabhbagchi/books-dataset)                    |
| merged_books_ratings.csv   | Contains merged book ratings and metadata | [merged_books_ratings.csv](https://www.kaggle.com/datasets/saurabhbagchi/books-dataset)   |
| ratings.csv                | Contains user ratings for books      | [ratings.csv](https://www.kaggle.com/datasets/saurabhbagchi/books-dataset)                 |
| Users.csv                  | Contains information about the users | [Users.csv](https://www.kaggle.com/datasets/saurabhbagchi/books-dataset)                   |
